# Under-the-Hood
A journey to become a developer who can explain why.

AI 없이도 설명할 수 있는 개발자가 되기 위한 기록.

보이는 결과보다,
그 안의 원리와 사고를 이해하고 싶다.

## Why

개발자로 일하고 있지만,
AI와 검색에 의존하는 순간이 많았다.

문제를 해결할 수는 있지만,
왜 그렇게 해결되는지 설명하는 데는 자신이 없었다.

그래서 다시 기초부터 시작한다.

이번에는 외우지 않고,
확실히 이해하면서.

## Learning Areas

- Java
- Spring
- Database
- Network
- Docker
- React
- Recommender System
- ...

## Motto

Make it work.  
Understand why.  
Explain clearly.
